# Design Notes - OceanFront Horizon Landing Page

## Tham khảo: Hà Sơn Tower (hasontowergelex.com)

### Tổng quan phong cách
- **Color palette:** Navy/dark blue (header, hero) + cream/ivory background + gold accents (#D4AF37 / #C9A961)
- **Typography:** Serif heading (Playfair Display) + Sans-serif body (Inter/Be Vietnam Pro)
- **Layout:** Section blocks rộng, padding lớn, thoáng đãng
- **Hero:** Ảnh nền tối, overlay gradient, CTA màu vàng đồng nổi bật
- **Cards:** Border subtle, shadow nhẹ, icon gold màu

### Cấu trúc các sections
1. **Header sticky:** Logo trái + nav menu + nút hotline gold
2. **Hero:** Full-screen image + badge "Sắp mở bán" + heading lớn + tagline + 3 stats inline + 2 CTA
3. **Overview:** Eyebrow text + heading serif + sub-text → 2-col layout (image + info table) → 6 highlights grid 3-col
4. **Location:** Dark navy background → eyebrow + heading + map image + danh sách kết nối hạ tầng
5. **Amenities:** Light bg + 2 ảnh feature lớn + 6 cards grid
6. **Promotions:** 3 cards lớn, gold accent + CTA dải gold giữa section
7. **Gallery:** Grid 4 cột với hover overlay "Xem ảnh"
8. **Contact:** 2-col (form + info liên hệ + Zalo/Phone/Email)
9. **Footer:** Dark navy, 3-col, copyright

### Floating CTAs (luôn hiển thị bên phải)
- Zalo (xanh) 
- Form đăng ký (vàng gold)
- Gọi (cam đậm)

### Cho OceanFront Horizon
- Giữ phong cách tương tự nhưng thay đổi color palette để phù hợp với chủ đề biển:
  - Primary: Deep navy ocean (#0A1F3A)
  - Accent gold: #C9A961 (sang trọng) hoặc teal-blue (#1E5F7E) cho variation
  - Background: Cream/ivory (#FAF7F2)
  - Hero overlay: Gradient từ navy đậm xuống transparent
- Thêm VR360 section (mới so với Hà Sơn)
- Hero: Dùng ảnh `11_Vewtongthe_TP_Final` (view tổng thể Trần Phú)
